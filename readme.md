# PSModule


You can think of Plaster as [Yeoman](http://yeoman.io) for the PowerShell community.

## Installation

If you have the [PowerShellGet](https://msdn.microsoft.com/powershell/gallery/readme) module installed
you can enter the following command:

```PowerShell
Install-Module PSModule -Scope CurrentUser
```

Alternatively you can download a ZIP file of the latest version from our [Releases](https://github.com/52-devops)
page.
